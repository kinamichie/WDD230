  • Calling a method is the same as invoking a function
  • An operator can be used to see whether an object has a particular property
  • To find all of an objects property, you can use an in loop
  • New properties and methods can be added to an object at any time
  • Properties do not always appear in the order they were entered
  • The names of object properties can be changed at anytime also
  • Object properties can be deleted
  • Objects that contain other objects are called nested objects
  • An object literal can be passed as a parameter to a function
  • The keyword "this" refers to an object that is within

  • Date objects can be used to create dates in different forms
  • Getter methods are able to return information

  • RegExp = a regular expression
  • Character groups can be placed together inside square brackets
  • In a regular expression, there are a number of characters that have a special meaning, commonly known as metacharacters:
    • . matches any character, except line breaks
    • \w matches any word chrarcter, and is equivalent to [A-Za-z0-9_]
    • \W matches any non-word character, and is equivalent to [\^A-Za=z0-9_]
    • \d matches any digit character, and is equivalent to [0-9]
    • \D matches any non-digit character, and is equivalent to [^0-9]
    • \s matches any whitespace character, and is equivalent to [ \t\r\n\f]
    • \S matches any non-whitespace character, and is equivalent to [^ \t\r\n\f\]
  • Modifiers can be placed after a token to deal with multiple occurrences of that token
    • ? makes the preceding token in the regular expression optional
    • * matches one or more occurrences of the preceding token
    • + matches one or more occurrences of the preceding token
    • {n} matches n occurences of the preceding token
    • {n,} matches at least n occurrences of the pattern
    • {,m} matches at most m occurrences of the preceding token
    • {n,m} matches at least n and at most m occurnces of the preceding token
    • ^ marks the position immediately before the first character in the string
    • $ marks the position immediately after the last character in the string
    • Any special character or modifers can be escaped using the backslash.
  • Greedy modifers will match the longest possible string
  • Lazy modifers match the shortest possible string
  • String methods: split(), match(), g fag, search(), replace()

  • DOM is not actually part of JavaScript
  • DOM shortcut methods
    • Document.body returns the body element of a web page
    • Document.images returns a node list of all the images in the document
    • Document.links returns a node list of all the "a" elements and "area" elements that have an href attribute
    • Document.anchors returns a node list of all the "a" elemments and "area" elements that have a name attribute
    • Document.forms returns a node list of all the forms in the document
