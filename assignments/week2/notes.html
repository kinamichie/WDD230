    Arrays, Logic, and Loops

  • Arrays are ordered lists of values
  • Create an array literal using square brackets that contain intitial values
  • When deleting an item from an array, it will leave an undefined value
  • Deconstructing an array is taking values out and presenting them as individual
  • Length of an array list use .length
  • To remove the last value of a list use .pop()
  • To remove the first value of a list use .shift()
  • To merge an array use .concat()
  • The join method turns the array into a string .join()
  • To find middle values of an array use .slice()
  • To remove items from an array and replace them use .splice(). This is a permanent method
  • Reverse order by useing .reverse()
  • Sort an array with .sort(). This is permanent puts it in alphabetical order
  • To find a particular value use .indexOf() or .includes()
  • Multidimensional arrays are an array of arrays
  • A set is a data structure that represents a collection of unique values. Cannot include any duplicate values
  • Create a set use new Set()
  • To add values to a set use .add()
  • To see if a value is in a set use .has()
  • To remove a value from a set use .delete() or to remove all values use .clear()
  • Sets can be converted into arrays
  • Maps can help keep track of inheritences so uneeded data can be deleted
  • Create a map using new Map()
  • Add entries to a map using .set()
  • Maps can be converted to nested arrays
  • Logic statements
    • if
    • else
    • Ternary operator is used as a shorthand to if, else statements
    • switch
    • while loops
    • do while loops
    • nested for loops
    • looping over arrays
    • looping over sets
    • looping over maps

    Functions

  • Functions are a chunk of code that can be referenced by name
  • Define a function literal using a function declaration, an anonymous function(), or a function constructor new Function()
  • To invoke a function enter it's name followed by parentheses
  • All functions return a value
  • Set parameters for your functions
  • Default parameters should always come after non-default parameters
  • Arrow functions use =>
  • Variable declarations that use the var keyword are automatically moved to the top of the current scope
  • callback() parameters make functions more flexible
  • Array iterators are forEach(), map(), reduce(), split(), filter(), reduceRight(), every(), and some()
  • Irators can be chained together
