Notes For Week 1

Master Mobile UX

  • Design for mobile first.
    • Address constraints of mobile size/bandwidth
    • Plan for future and make ready for growth
    • Understand many households do not have desktop computers anymore
    • Start small and scale up
    • Mobile is not a fad
  • How to Improve Mobile UX by Understanding Users
    • Be there
    • Be Useful
    • Be quick
    • Reachability matters
    • Speed matters
  • Better Mobile UX With Strategic Menu Design
    • Move navbar to bottom
    • Always strive for amazing
    • "menu" with a border was found best
  • Designing Intuitive and Mobile Friendly Forms
    • Make it simple so people can use it easily
    • Display labels inside fields
    • Break forms into sections
    • Enable auto-fill
    • Show passwords but give the user the option to hide them
    • Easy credit card entry
    • Calls matter
  • Designing a Better Mobile Advertising User Experience
    • Research
    • Stop using pop-ups
    • Users don't visit your website to look at ads
    • Keep it simple and sleek
    • Make sure it's unobtrusive; content first
  • Ask: What would I personally think?
